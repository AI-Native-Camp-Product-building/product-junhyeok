# Ecosystem Scan — 소스 상세 가이드

ecosystem-scout가 참조하는 상세 소스 목록과 접근 방법.

## 공식 소스

### Anthropic 블로그
- URL: anthropic.com/news 또는 anthropic.com/blog
- 주요 콘텐츠: 제품 업데이트, 기능 발표, 기술 글
- 주기: 비정기 (주요 릴리즈 시)

### Claude Code 문서
- URL: docs.anthropic.com/en/docs/claude-code
- 주요 콘텐츠: 기능 가이드, API 레퍼런스, 튜토리얼
- 특히 확인: /changelog, /release-notes 섹션

### GitHub anthropics/claude-code
- URL: github.com/anthropics/claude-code
- 주요 콘텐츠: 릴리즈 노트, 이슈, 디스커션
- 확인 대상: Releases 탭, Discussions 탭, 인기 이슈

## 커뮤니티 소스

### Reddit
- r/ClaudeAI — 일반 Claude 관련 (Claude Code 필터링 필요)
- r/ChatGPTCoding — AI 코딩 도구 전반 (Claude Code 포함)
- 정렬: Top (이번 주/이번 달) → Claude Code 키워드 필터

### X/Twitter
- 검색 쿼리: `"claude code" min_faves:10` (최소 좋아요 필터)
- 팔로우 대상: @AnthropicAI, Claude Code 관련 개발자
- 해시태그: #ClaudeCode, #ClaudeDev, #AIcoding

### YouTube
- 검색: "claude code tutorial" 또는 "claude code workflow"
- 정렬: 최근 업로드 → 조회수 순
- 10분 이상 영상이 보통 실질적 콘텐츠를 포함

### 개발자 블로그
- Medium, dev.to에서 "claude code" 검색
- HackerNews에서 "claude code" 검색 (comments 포함)

## 생태계 소스

### MCP 서버
- GitHub 검색: `mcp-server` topic, 최근 업데이트 순
- awesome-mcp-servers 같은 큐레이션 레포
- Anthropic 공식 MCP 레포

### IDE 통합
- VS Code Marketplace: Claude 관련 확장
- JetBrains Marketplace: Claude 플러그인

## 검색 실패 시 대체 전략
1. 검색어 변형: "claude code" → "claudecode" → "claude AI coding"
2. 기간 확장: 1주 → 1달 → 3달
3. 소스 확장: 영어 외 일본어/한국어 커뮤니티도 탐색
