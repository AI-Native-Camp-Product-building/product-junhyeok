## Key Decisions
- **L1(in-memory) → L2(fs/Blob) → L3(pipeline) 계층화**: serverless cold-start + 서버 재시작 양쪽 해결
- **Vercel Blob + filesystem 이중 백엔드**: `BLOB_READ_WRITE_TOKEN` 유무로 자동 전환, dev 무설정 동작
- **Cron schedule 23:00 UTC = 08:00 KST**: 9시 브리핑 1시간 전 warm
- **~/.zshrc 5개 키 guard 패턴**: Keychain 빈 export → dotenv override 차단 근본 원인 차단
- **mock-feed-data.ts 함수화**: `getMockFeedItems()`에서 KST 상대 날짜 동적 생성
