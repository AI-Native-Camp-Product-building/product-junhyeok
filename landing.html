Quick Start

1분이면 시작할 수 있습니다

terminal
# 플러그인 심링크 생성
$ ln -sf ~/workspace/ai-tools/claude-dopamine-sprint \
~/.claude/plugins/local/claude-dopamine-sprint
# Claude Code에서 바로 사용
$ claude
> /sprint
# 또는 원커맨드 실행
$ ./scripts/study-claude.sh

명령어 레퍼런스

/sprint [토픽ID] 30분 학습 스프린트
/sprint mini 15분 미니 스프린트
/quiz [토픽ID|all] 5분 퀵 퀴즈
/streak 연속 학습일 + 업적
/dashboard 전체 진행률 대시보드
/sprint update 새 기능 자동 감지 + 토픽 추가
/reset [scope] 진행률 리셋
